日志快速查询系统

2015.08 - 2016.01

  • 项目描述：基于 HBase 开发常用字段的快速查询系统
  • 工作内容
    1. 改写 InputFormat，改进后 MR 作业读取 csv 文件的平均 Map 时间节省了60.6%，读取 lzo 压缩 文件的平均 Map 时间节省了69.1%；
    2. 开发 Kafka 客户端，代替运营商省网项目中 importer，将数据传入到 hdfs 和 HBase，该运营商每天的数据量大约是1.2TB；
    3. 利用 HBase 基于 RowKey 的快速查询为 hdfs 上的文件话单建立常用字段索引表。

网络流量流向分析系统

2015.02 - 2015.07

  • 项目描述：将实验室的实时网络访问信息以地图的形式进行展示。
  • 工作内容
    1. 设计和搭建实验室数据实时处理平台，使用 Flume 和 Kafka 实时监控实验室话单；
    2. 完成基于 Storm 的流处理模块开发；
    3. 在流处理模块中设计了数据聚合算法。

Kafka 性能测试与优化

2014.09 - 2015.05

  • 项目描述：测试和优化 Kafka 性能，设计并实现流处理系统。
  • 工作内容
    1. 测试和优化 producer、consumer 性能，优化后 producer 速度提高14%，consumer 速度提高21%；
    2. 研究 Kafka 与 Storm、Spark Streaming 的集成，分别实现一个流量实时统计系统;
    3. 在 IHMSC2016 会议上以第一作者的身份发表一篇关于 Kafka 性能优化的论文。
