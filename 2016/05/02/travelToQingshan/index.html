Travel to Qingshanbei | 河北青山背露营

发表于 2016-05-02   |   分类于 travel   |     |   次阅读

写在前面

本来是不打算把旅行户外的游记写在博客上，后来想想为了维护方便，还是写在博客上吧。这次跟着一个户外俱乐部参加了户外露营的活动，这是我第一次去重装爬山在山里露营，之前虽然也爬过很多的山，但是从来没有重装爬过，这也算是对自己的一次挑战吧。之所以会选择这次活动，出发点有很多，想去看看星空、日出、云海，当然结果是基本都没看到， :joy: ，期望太高了，选择这次活动还有另外一个原因，今年是找工作的一年，在找工作之前需要找一个实习，这样暑期的时候可以提前去公司体验感受一下，实验室的好多小伙伴现在基本上都有offer在手了，甚至有些人拿到了几个大公司的offer，而我现在可以说到手的一个offer都没有，腾讯连给面试机会都没有( :sob: )，所以也就想趁着五一出去散散心，因此就有了这一次的经历。

活动报名

这次的报名主要看到了一个领队发的一个活动连接，当时的连接上写的是北灵山露营，看着还不错，价钱也合适（活动费200，露营装备60），所以就报名了，之前跟过这个领队参加过一次活动，所以加过他的微信。没想到最后活动就变成了青山背，他们说灵山的风景一般，当时感觉都可以吧，对具体去哪座山要求不多。关于这些户外活动的信息，最大的网站应该就是绿野户外了，因为上面的活动可能来自很多的户外组织，所以还是需要大家自己去鉴别的，具体怎么鉴别看个人的经验吧。如果是在北京上学的学生或者年轻人的话，推荐草履虫户外，这个还是很靠谱的，很多的领队都是学生或者刚毕业的童鞋，而且参加活动的人员基本上也有一半是学生吧，比较偏年轻化。

4.30

因为7点25集合，所以早上很早就起床了，到达集合地点之后，7点半左右，我们就做着小大巴出发了。今天是五一的第一天，你懂的，出北京的高速是必堵了，果然不出所料，在延庆那边堵了有两个小时，堵车的原因一个是收费站缴费（收费站也确实太多了），另一个就是那条路发生了两起车祸，两辆小三轮追尾了，另一个是两辆私家车追尾了，因为是双向单行道，所以就导致了整条路都堵了（出去玩的话还是安全第一）。到了下午两点，我们达到了一个村庄，就是我们目的地前面的一个村庄，我们在小卖部那边添加了一些补给。这个村庄基本上都是瓦房，而且是那种有炕的瓦房，这种房子我之前没有见过。

达到目的地之后，山旁边有个寺庙，那边有一块大平地，可以停车，我们在那里弄好装备之后，就开始出发。我们一行二十来人（一看我就不是专业的，最右边 :blush: ）

我们开始重装，沿着山谷前行

感觉这个峡谷，之前像是一条河流，在这条峡谷上，前面会有很多的石头。走了一段之后，我们开始向右边的山攀爬，山路还是很陡峭的，坡度比较大，而且有些地方还是很难站稳，这时候一定注意，每一步都要踩稳，毕竟我们都是背着重装前行的，本来重心就不是太稳。

站在不同的角度，风景就会有很大的不同，下面放几张美景

这时候，命运就开始捉弄我们了，因为山路一直很陡，天也越来越晚，所以有人就建议，领队先上去，看看有没有地方可以扎营，如果没有的话，我们只能返回到山脚去扎营，因为天气预报会有雨，所以必须找一个安全的地方去扎营。最后结果是原路返回，回到停车场去扎营， :joy: 这么陡峭的山，上山容易下山就难了，毕竟大家还都是重装，但是也没办法，只能回去

Tips: 如果是新手，第一次去露营的话，建议选择成熟路线，尽量选择风景比较好设施也比较好的成熟路线去露营，虽然这些地方商业化比较严重，但是为了安全着想，还是稳妥一些比较好。（户外老司机可以忽略哈）

达到营地后，大家开始扎营

傍晚的寺庙

感想：晚上的风超级大，感觉分分钟帐篷要被刮走的节奏，而且帐篷里会落很多的土，比较脏，因为是在山脚扎营，所以晚上并不是很冷，甚至还有点热。

5.1

早上当然是看不到日出的，这里也不是观看日出的地方。早上吃完饭之后，我们就开始轻装再次向山里进军

路途中的一些合照

路上还能看到一些冰，这些冰应该是常年不化的

途中的瞬间

山上有很多的挑花树，突然就让人想到了白居易的诗“人间四月芳菲尽,山寺桃花始盛开”

我们连续爬了三个多小时，这三个多小时的路真的可谓是“劈荆斩棘”，因为进山没有什么路，基本上都是看着哪边好走就走哪边，旁边有很多草很多树，有的会带刺会扎人，总之，这一路真的是很不容易。最后，我们达到了一个小瀑布那里，瀑布已经结成了冰

再换个角度看这些山：

我们浩浩荡荡又开始下山

回到山脚

再次回到来时的那个寺庙

最后

这是我人生中第一次露营，总体感觉，没有想象中那么好，但是这是我进入户外的又一个开始、这是我露营的一个开始，以后我肯定还会继续去挑战一些更高度的户外活动。
